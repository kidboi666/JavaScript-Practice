1.기본문법
 참조 사이트 : 생활코딩
 참조 서적 : 
 개요 : 자바스크립트로 프로그래밍을 하기 위한 최소한의 기본 문법을 익히는 목적으로, 기본적인 자료형과 함수, 객체, 배열의 기본 개념들을 보고 듣고 느낀 부분을 모두 저장했다.
 다만 저장소를 개설한 시점에선 이미 여러번 훑어본 시점이라, 개념 확립 차원에서 찬찬히 커밋할 예정
